# Contributing to The Mindful Trader

Thank you for your interest! This is currently a personal project, but feedback and suggestions are welcome.

## Reporting Issues

If you encounter a bug or issue:

1. Check if the issue already exists in GitHub Issues
2. If not, create a new issue with:
   - Clear, descriptive title
   - Steps to reproduce the problem
   - Expected behavior vs actual behavior
   - Screenshots if relevant
   - Your environment (browser, OS, Node version)

## Suggesting Features

Have an idea for a new feature?

1. Open an issue with "Feature Request" label
2. Describe the use case clearly
3. Explain how it would help traders
4. Consider if it fits the psychology-first focus

## Code Contributions

While this is a personal project, I may consider pull requests for:
- Bug fixes
- Documentation improvements
- Performance optimizations
- UI/UX enhancements

### Before Contributing Code

1. Open an issue to discuss the proposed change
2. Wait for feedback before investing significant time
3. Follow the existing code style and conventions

### Code Style

- TypeScript strict mode enabled
- ESLint rules enforced
- Prettier for code formatting
- Meaningful variable and function names
- Comments for complex logic

### Development Workflow

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Test thoroughly locally
5. Commit with clear messages
6. Push to your fork
7. Open a pull request

### Testing Checklist

Before submitting a PR, ensure:
- [ ] Code builds without errors (`npm run build`)
- [ ] No TypeScript errors (`npm run type-check`)
- [ ] ESLint passes (`npm run lint`)
- [ ] Tested in development mode
- [ ] Tested with actual trading data
- [ ] Mobile responsive (if UI changes)
- [ ] Works in Chrome, Safari, Firefox

## Questions?

Have questions about the project or how to contribute?

- Open a GitHub Discussion
- Check existing documentation first
- Be specific and provide context

## Code of Conduct

Be respectful, constructive, and professional. This project aims to help traders improve, so keep that goal in mind.
